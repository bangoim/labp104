# labp104

Implementacao completa de um Transformer (Encoder-Decoder) do zero usando NumPy. Inclui atencao escalada, mascaramento causal, FFN, inferencia auto-regressiva e todos os blocos do paper "Attention Is All You Need".

## Como rodar

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

```bash
python3 tests/test_transformer.py
```
